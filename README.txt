[KR] 빠른 사용법
1) index.html, styles.css, script.js 파일의 문구를 본인 프로젝트에 맞게 수정합니다.
2) GitHub에 새 저장소를 만들고 이 3개 파일을 업로드하세요.
3) Vercel(https://vercel.com)에 로그인 → Add New Project → GitHub 저장소 선택 → Deploy.
4) 커스텀 도메인은 Project → Settings → Domains에서 연결합니다.
5) 문의 폼: Formspree(https://formspree.io)에서 폼을 생성하고, index.html의 form action URL을 교체하세요.

[EN] Quick start
- Edit text in index.html, tweak colors in styles.css if needed.
- Push to GitHub, then import to Vercel and Deploy.
- Replace the contact form action with your own Formspree endpoint.
